Formation Full Stack avec Java Entreprise Edition (JEE) – ENSIAS

Devenez Développeur Full Stack JEE en 16 semaines avec une formation intensive adoptant l'approche "Learning by doing" et un accompagnement à l'insertion professionnelle.

Pourquoi devenir Développeur Full Stack JEE ?

Le développeur Full Stack est un profil recherché par toutes les entreprises. C'est l'une des fonctions IT les mieux rémunérées et les plus recherchées par les RH. Ce profil polyvalent est en mesure de gérer un projet web de bout en bout, travaillant sur la partie visible de l'interface web (front-end) et sur la partie technique (back-end).

  • Demande élevée
  • Salaire compétitif
  • Possibilité d'évolution de carrière
  • Défis techniques
  • Collaboration avec d'autres professionnels

Méthode d'Enseignement

La formation adopte la méthode "Learning By Doing", basée sur la pratique, l'apprentissage en groupe et une approche d'apprentissage par projet.

Ce que vous allez apprendre

Cette formation vous dotera des concepts nécessaires pour mettre en place des solutions Full Stack en utilisant Java pour le back-end et JavaScript pour le front-end. Vous acquerrez également les outils et les compétences nécessaires pour gérer des projets agiles, communiquer et collaborer avec les parties prenantes. De plus, l'accent sera mis sur l'application du Design Thinking pour produire des solutions innovantes et compétitives.

Programme de Formation

  • Module 1 : Les bases de la programmation en Java
  • Module 2 : POO & UML
  • Module 3 : Java EE (JSP, Servlet, ...)
  • Module 4 : CyberSecurity
  • Module 5 : JavaScript

Postes après la Formation

  • Développeur Full Stack JEE
  • Développeur Web
  • Développeur de logiciel
