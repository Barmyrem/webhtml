Module 3 : Java EE (JSP, Servlet, ...)

Ce module explore Java EE (Java Platform, Enterprise Edition) et couvre les technologies de développement Web telles que JavaServer Pages (JSP) et Servlets.

Thèmes abordés :

  • Introduction à Java EE
  • JavaServer Pages (JSP) pour la création de pages web dynamiques
  • Servlets pour le développement d'applications web côté serveur
  • Modèle MVC (Modèle-Vue-Contrôleur) pour la conception d'applications web

Objectifs d'apprentissage :

  • Comprendre Java EE et ses composants pour le développement d'applications web d'entreprise.
  • Maîtriser l'utilisation de JavaServer Pages (JSP) pour la création de pages web dynamiques.
  • Savoir développer des Servlets pour le traitement des requêtes HTTP.
  • Appliquer le modèle MVC pour concevoir des applications web bien structurées.
